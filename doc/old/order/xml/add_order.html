2. Oktober 2002

XML-Kommando <add_order>  -  Neuen Auftrag erzeugen

Über das XML-Element <add_order> kann ein Auftrag an eine Jobkette übergeben werden. Die Attribute state, priority, id und title sind optional. Die Parameter werden in order.payload als Variable_set bereitgestellt. 

<add_order job_chain="name" state="state" priority="priority" id="id" title="title">
    <params>
        <param name="param1" value="wert1"/>
        ...
    </params>
</add_order>


Ein Auftrag kann auch direkt einem Job zugestellt werden. Die Attribute job_chain und state werden dann durch job ersetzt. 

<add_order job="name" priority="priority" id="id" title="title">
    <params>
        ....
    </params>
</add_order>


Die id wird immer als String übernommen. Ein Auftrag mit der String-id "100" und einer mit der numerischen id 100 (mit add_order() eingefügt) sind nicht gleich.

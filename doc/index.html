Scheduler

factory.ini

Einige Einstellungen können in der Datei factory.ini vorgenommen werden.

XML-Konfiguration

Die XML-Konfiguration beschreibt die Jobs und alles was dazu gehört.

Kommandozeile

Start mit der Kommandozeile

Bedienung über HTTP

Der Scheduler verfügt über einen HTTP-Server und stellt HTML-Seiten zu seiner Bedienung bereit. Sie können den Scheduler über einen Browser (zurzeit nur Microsoft Internet Explorer 6) steuern. Geben Sie z.B. die Adresse http://localhost:4444 ein. localhost kann auch ein anderer Rechnername sein. 4444 ist die konfigurierte TCP-Portnummer des Schedulers.

Bedienung über telnet

Die XML-Kommandos können auch über einen telnet-Client eingegeben werden. Aufruf: telnet localhost 4444.

Protokolle

Im mit log_dir eingestellten Verzeichnis schreibt der Scheduler seine Protokolle.

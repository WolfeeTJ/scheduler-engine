        : op = 'U'? 'UPDATE' 
        : op = 'S'? 'STORE' 
        : op = 'D'? 'DELETE' 
        :           ''

FROM 'fs:isam.datei' [DELETE] [ERROR error_flag = 'E']

Schlüsselwort DELETE in FROM-Klausel löscht jeden gelesenen Satz,
wenn er erfolgreich in die Ausgabedatei gespeichert werden konnte (nicht bei Join).

ERROR feld = value gibt an, wie fehlerhafte Sätze markiert werden sollen. 
Tritt beim Markieren eines fehlerhaften Satzes ein Fehler auf, wird die Replikation abgebrochen.

(die ersten hundert in einer Speicherdatei)
und löscht sie bei COMMIT oder bei CLOSE NORMAL.
